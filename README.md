# Getting Started:

1. You should probably git pull from main.
2. You need to get python 3.12.6.
3. **python3 -m venv .venv**
4. **source .venv/bin/activate**
5. **python3.12 -m pip install -r requirements.txt**
6. You should probably git pull from main
7. Work on code changes.
8. **git commit -m “Commit message here.”**
9. git push
10. **deactivate**
